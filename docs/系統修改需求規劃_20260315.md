## 一、人員技能與人員管理

### 1.1 技能分類調整
- 原「技能」改為「系統安裝技能」
- 新增「通用能力」分類：應對溝通協調能力、施工速度、查修能力、領導統籌能力

### 1.2 對講相關技能
- 原「對講」細分為：
  - Hometake對講系統安裝查修
  - BBhome對講系統安裝查修
  - 大華對講系統安裝查修

### 1.3 監控相關技能
- 增加：類比系統安裝維修、大華數位系統安裝維修、海康數位系統安裝維修、宇視數位系統安裝維修、TPLink系統維護/安裝/維修

### 1.4 其他技能
- 「原技能設定」改為「設備安裝技能」
- 管線改管路施工，增加：壓條施工、RC穿牆、鋁製線槽配置、吊管工程、切地埋管、架空作業
- **技能系統管理者可新增、編輯、刪除**

### 1.5 人員管理
- 配對表放在技能設定旁，將同公司其他人員列出，集中做配對設定
- 人事行政中的人員管理暫時不動，作為後續公司人事管理用

## 二、工程日曆／排工管理

### 2.1 行事曆呈現
- 以整月日曆顯示
- 當日案件過多時，最後顯示「+N」，點擊後懸浮放大顯示所有案件，可點案件查看內容
- 單日用顏色區分：**綠色=可排、黃色=部分已滿、紅色=已滿**
- 登入者的案件顯示在上方，用顏色區分（業務看自己的案件、工程人員看自己的排工）

### 2.2 新功能
- 右上「新增案件」旁增加「**依人員查詢**」按鈕
  - 點入後可選擇業務或工程人員
  - 用月份顯示該人員負責案件
  - 可點選某日查看詳細
- **第幾次施工**由系統自動計算，不可手動選擇
- 點擊施工次數可跳出歷史排工目錄，可進一步查看當次實際施工內容

### 2.3 收款與時間
- 回報功能增加「**有無收款**」選項，若有收款必填金額
- 記錄**到達時間**與**離開時間**
- 評估加入「**預計下次施工時間**」

### 2.4 排工權限
- 有排工權限者可編輯工程日曆
- 無排工權限者只能查看，不會出現右上新增排工按鈕
- 工程人員原則上只能看，但可點擊進入回報功能

### 2.5 鎖定邏輯
- 當日工程已滿後，除主管或有權限管理者外不可再更動

### 待確認

- [ ] 預計下次施工時間是否要做？

## 三、工程回報與施工紀錄

### 3.1 入口
- 工程師可用的按鍵中增加「工程回報」按鈕

### 3.2 回報方式
- 支援新增**照片、文字**，可多次施工回報
- 呈現方式：**延續式/時間軸式**，可上下滑動查看歷史
- 每日回報有明確日期區分（如 3月15日、3月16日）

### 3.3 回報內容
- 文字與圖片
- 增加「**使用器材**」與「**使用耗材**」選單
  - 支援勾選項目、輸入數量
  - 可新增細項
  - 能計算成本
- 評估後續可否由出貨單直接帶入使用器材資料

### 3.4 上下工提醒
- 施工結束後需按回報選單
- 可填**上工時間**與**下工時間**
- 未填系統顯示個人提醒
- 個人可看自己的提醒；主管可看所有人的提醒

- [ ] 器材/耗材選單從產品目錄帶入？還是獨立清單？
- [ ] 出貨單系統目前是否存在？

## 四、案件管理與維修單管理

### 4.1 案件附件
- 案件管理內可上傳檔案與圖片
- 上傳類型至少包含：**施工圖、報價單、保固書、預計使用線材**
- 各欄位/區塊需有上傳按鈕

### 4.2 案件難易度
- 除業務自行判斷外，增加「**系統自動判斷**」結果
- 系統依業務輸入資料評估，可與業務判斷做比對

### 4.3 案件管理新增欄位（排工相關）
- 預計施工日
- 預計完工日
- 是否可隨時安排（當日有狀況時可調度先行施工）
- 施工時間
- 有無施工時間限制
- 客戶休息時間
- 是否可夜間加班
- 以上欄位需可做查詢篩選

### 4.4 案件列表 UI 改善
- 狀態與類型改用按鈕篩選（非下拉選單）

### 4.5 維修單管理
- 比照工程管理，增加回報、圖片上傳、文字紀錄功能

### 待確認
- [ ] 檔案上傳到主機空間？還是雲端？（智邦主機空間限制）
- [ ] 系統自動判斷難易度的算法依據？

## 八、智慧排工

### 8.1 入口
- 案件管理首頁操作旁增加「智慧排工」按鈕，可直接安排工程
- 已在工程行事曆裡的案件需顯示是由誰排工或是智慧排工

### 8.2 排程限制
- 當日工程已滿，除主管外不可再更動
- 大型案件需考量人力資源調度，不可只做單一案件排程

### 8.3 智慧判斷條件
- 案件急迫性
- 後續安排、完工日前人員進場需求
- 大型案件需判斷需要多少人力、多少天完成
- 驗收日是否也要派工
- 當日有工程人員完成其他施工且有空檔，可安排前往支援大型案件
